Development

These are the various applications (for desktop, web and mobile) that I have developed or played a part in developing so far. Most of these have source code and executable files available on request via the contact section. I am currently setting up on github and hope to have these projects available as soon as possible.

Stream

Stream is an experiential puzzle game (desktop app) that is designed to educate students about the nature of fields and field mechanics such as those of gravitational, electric and magnetic fields. The game is developed using Java and features a particle system, a physics engine and 15 playable levels with user generated content.

Dash

Dash is a vehicle game (desktop app) designed and developed for the FBLA (Future Business Leaders of America) computer game simulation contest. The game features a timed obstacle course with objects designed in Maya and implemented in Unreal Engine 4 using blueprint scripting and C#

Shift

Shift is a stunt game (desktop app) developed in Unreal Engine 4 that is designed to teach high school and college students concepts about basic economics. The game is an obstacle course designed from graphs and shapes related to economic concepts. Each checkpoint to be passed contains reviews and short quizzes for the player to complete.

Res Novae

Res Novae is the website of a STEM focused student publication developed with a bootstrap frontend and a wordpress backend. The website contains several articles about cutting edge technology, research and science related news.

CycleFitness

CycleFitness is a website developed for a hypothetical bicycle shop in New York as a part of a website design contest. The website contains an HTML/CSS frontend which is facilitated by an aspx backend developed in Microsoft Visual Studio.

Timezone

Timezone (tentative) is an android mobile app designed to allow users to collaborate meeting and schedules from across timezones. Timezone was developed during an internship at Clarice Technologies. The app is currently being developed in Android Studio using Java.
